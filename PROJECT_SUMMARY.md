# 场外基金投资辅助工具 - 项目总结文档

## 项目概述

场外基金投资辅助工具是一个基于Flask和SQLite的后端项目，旨在为用户提供便捷的基金投资管理功能。项目涵盖了用户认证、基金数据管理、自选功能、市场行情、交易功能、消息中心和个性化设置等核心模块。

## 技术架构

- **后端框架**: Flask 2.x
- **数据库**: SQLite 3
- **数据库ORM**: SQLAlchemy
- **API文档**: Flask-RESTX
- **用户认证**: Flask-JWT-Extended
- **数据验证**: Marshmallow
- **文件上传**: Flask-Uploads

## 核心功能模块

### 1. 用户认证模块
- 用户注册、登录、登出
- JWT身份验证
- 用户资料管理
- 密码加密存储

### 2. 基金数据模块
- 基金列表查询（支持分页、筛选）
- 基金详情查看
- 基金搜索功能
- 基金历史净值查询

### 3. 自选功能模块
- 自选基金增删改查
- 自选分组管理
- 分组内基金管理
- 批量操作功能

### 4. 市场行情模块
- 市场基金列表
- 热门板块展示
- 市场资讯功能
- 主要指数查询
- 板块预测功能

### 5. 交易功能模块
- 持仓管理功能
- 交易记录功能
- 买入/卖出模拟功能
- 资产概览功能

### 6. 消息中心模块
- 消息列表功能
- 消息状态管理
- 消息推送机制

### 7. 设置模块
- 用户设置管理
- 缓存清理功能
- 个性化设置

## 数据库设计

项目使用SQLite数据库，包含以下核心表：

- users: 用户信息表
- user_profiles: 用户资料表
- user_settings: 用户设置表
- funds: 基金基础信息表
- fund_market_data: 基金行情数据表
- fund_groups: 基金分组表
- favorite_fund_relations: 用户自选关系表
- holdings: 用户持仓表
- transactions: 交易记录表
- notifications: 消息通知表

## API接口设计

项目使用RESTful API设计风格，所有接口均通过Flask-RESTX进行文档化。API接口包括：

- 用户认证接口（注册、登录、资料管理等）
- 基金数据接口（查询、搜索、详情等）
- 自选功能接口（增删改查、分组管理等）
- 市场行情接口（基金列表、板块、资讯等）
- 交易功能接口（持仓、买卖、记录等）
- 消息中心接口（消息列表、状态管理等）
- 设置接口（用户设置、缓存清理等）

在生产环境中，建议：

- 使用Gunicorn作为WSGI服务器
- 配置Nginx反向代理

- 配置更强大的数据库（如PostgreSQL）
- 设置定期数据库备份

1. **功能完整性**: 涵盖了基金投资的核心场景，从市场行情查看、基金自选管理到账户资产分析

2. **用户体验优化**: 个性化首页、直观的涨跌展示、便捷的数据导入方式

3. **设计一致性**: 统一的视觉规范、交互模式，提升整体使用体验

5. **便捷操作**: 支持批量操作、快捷入口、自定义设置等，提高用户操作效率
